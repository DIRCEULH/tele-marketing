📞 Sistema de Feedback – Telemarketing

Sistema desenvolvido para coleta, gerenciamento e análise de feedback de clientes em operações de telemarketing, permitindo melhorar a qualidade do atendimento e a tomada de decisões.

📌 Descrição

Este projeto tem como objetivo registrar e organizar o feedback dos clientes após interações com a equipe de telemarketing.

A solução permite acompanhar avaliações, identificar pontos de melhoria e gerar insights sobre o desempenho do atendimento.

⚙️ Tecnologias utilizadas
PHP
Angular
MySQL
API REST
